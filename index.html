Crystals Collector!

Rules

At the start of each game you'll be given a random number to match.

Each of the four crystals on the right has a mystery value between 1 and 12. Click a crystal to add it's value to your score.

If you match the value of the random number without going over, you win.

If you go over the random number you lose.

Each time you play the crystals will have new mystery values.
